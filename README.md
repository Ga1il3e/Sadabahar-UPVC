# Sadabahar UPVC Landing Project

Premium scrollytelling landing page for Sadabahar UPVC with a scroll-scrubbed frame sequence and 3D engineering reveal.

## Project Structure

- `index.html` - entry file (redirects to landing page)
- `sadabahar-upvc-landing.html` - primary landing page
- `assets/css/final-tweaks.css` - final contrast/readability and visual polish overrides
- `assets/js/finalize.js` - non-breaking runtime polish helpers
- `animation_gif/assembled_frames` - assembled sequence frames
- `animation_gif/exploded_frames` - exploded sequence frames

## Local Preview

From project root:

```powershell
python -m http.server 4173
```

Then open:

- `http://127.0.0.1:4173/`
- or directly `http://127.0.0.1:4173/sadabahar-upvc-landing.html`
